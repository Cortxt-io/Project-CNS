# cortxt-dashboard-app / roadmap

## v1 — MVP (pågår)
- React 18 + Vite + Tailwind + Reactflow
- Portföljgrid med filter (status, family)
- Projektdetaljsida med renderad Markdown
- Reactflow graph-overlay, noder per family, inga edges
- Deploy: Cloudflare Pages (app.cortxt.io)

## v2 — Planerat
- Edges i graph baserade på beroenden i project.md
- Family-sidor (en detaljsida per produktfamilj)
- MCP-server som kompletterande datakälla
- Drag-and-drop för att flytta projekt mellan familjer
- Scoring Studio-integration
